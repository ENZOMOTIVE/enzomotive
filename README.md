## About Me

I am a builder who likes turning rough ideas into working prototypes. My repositories move across full-stack web apps, WhatsApp and LINE agents, mobile wallets, decentralized applications, credential systems, and small learning projects that sharpen the fundamentals.

The common thread is practical experimentation. I use projects to test how an idea feels in the real world: how a user signs in, how a wallet connects, how an AI agent responds, how a credential is issued, how a dashboard explains the state of a system, and how a mobile app can carry that experience around.
